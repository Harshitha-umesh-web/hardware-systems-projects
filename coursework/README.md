# Graduate Coursework – M.S. Electrical & Computer Engineering  
*Northeastern University*

This page summarizes selected graduate-level coursework aligned with RF systems, mixed-signal IC design, high-speed data communication, sensing, and semiconductor devices.

---

## EECE 7242 – Integrated Circuits for Mixed Signals & Data Communication  
**Tools:** Cadence orCAD, LTSpice  

**Topics Covered:**
- Broadband analog and high-speed digital IC design  
- Differential signaling, current-steering architectures  
- Noise analysis and SNR optimization in ICs  
- Optical data links and BER performance  
- Frequency synthesis: PLLs, VCOs, CDR, phase noise & jitter  
- High-speed ADC/DAC architectures and implementation  
- Continuous-time and sampled-data IC filter design (Gm-C, switched-C)  
- PAM-4 and QAM/32-QAM modulation for high-speed links  

---

## EECE 5698 – Visual Sensing & Computing (Co-Design for Edge Machine Perception)  
**Tools:** Python  

**Topics Covered:**
- Computational optics and image formation  
- Optical-to-electrical signal conversion (image sensors)  
- Sampling and reconstruction principles  
- Noise sources and dynamic range in imaging systems  
- Image signal processing pipelines  
- Image/video compression fundamentals  
- Vision system co-design (optics + sensors + processing)  
- Hardware-aware edge machine perception  

---

## EECE 5665 – Signal Processing for GNSS  
**Tools:** MATLAB  

**Topics Covered:**
- GNSS signal modeling and synchronization  
- UTC time reference systems  
- Kalman filtering and state estimation  
- Timing drift and jitter analysis  
- Multi-constellation processing (GPS, Galileo)
